# E2EE Messaging App

An iOS application demonstrating End-to-End Encrypted messaging with real-time capabilities, built with SwiftUI and Swift.

## 🚀 Features

- Real-time encrypted messaging
- User authentication
- Chat channel management
- End-to-End Encryption (E2EE)
- iOS native support
